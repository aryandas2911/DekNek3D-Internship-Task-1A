# DekNek3D Conversion Service
A backend service for converting GLB 3D models into STL, OBJ, PLY, and 3MF formats.
Built as part of an internship assignment for DekNek3D.

---

## Overview
This service provides an HTTP API to upload .glb files, convert them into standard 3D formats using a Python-based pipeline, and download the converted output.

---

## Features
- Upload GLB 3D model files
- Convert to STL, OBJ, PLY, or 3MF
- Simple REST API
- Fast, on-demand file conversion
- Clean separation of backend logic

---

## Tech Stack
- Backend: Node.js, Express
- Conversion Layer: Python
- File Handling: Multer
- Deployment: Render
- Frontend: React
